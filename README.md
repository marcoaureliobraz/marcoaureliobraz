# 🌎 Marco Aurélio Braz

**`Estudante da USP | Iniciação Científica (IC)|`**

Sou graduando em Oceanografia pela **Universidade de São Paulo** com **interesse na aplicação de Ciência de Dados, Inteligência Artificial e Machine Learning à Oceanografia Física**. Atualmente, estou desenvolvendo minha base em Python para análise de dados científicos, automação e modelagem computacional, buscando compreender e interpretar os processos físicos do sistema  Estuarino-Lagunar de Iguape-Cananéia. Meu principal objetivo é integrar programação e ciência para desenvolver soluções baseadas em dados que contribuam para a análise, modelagem e previsão de fenômenos como **correntes oceânicas, massas d'água, interação oceano-atmosfera, ondas, marés, eventos extremos, riscos costeiros e outros processos físicos** do ambiente marinho.
